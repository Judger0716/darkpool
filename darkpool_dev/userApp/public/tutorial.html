暗池交易系统功能介绍及使用说明

注册与登录
Login&Register

点击首页中的“登录/注册”按钮进入登录/注册界面（如左图所示） 若已有本平台账户，则输入用户名和密码即可登录 若尚未有本平台账户，则点击注册按钮，注册账户并登录

立即登录/注册

订单查询
Order Query

在订单查询窗口，可以查询到系统中其他用户的挂单信息 在暗池交易模式下，您将无法查看其他用户订单的关键信息 但仍可以看到部分相关信息

创建订单
Create Order

在创建订单窗口，您可以在系统中提交自己的订单 输入订单的详细信息，系统将自动为您进行隐私保护工作，您的订单关键信息将不会显示在系统中，也无法被用户查询 挂单完成后，您可以在“订单查询”中查询到自己的订单

查询匹配交易
Dealed Order Query

在查询匹配交易窗口，您可以看到系统中最新成交的订单信息 包括买卖双方、交易物品和成交价在内的详细信息，于此同时，这些信息也将更新至相关查询中 通过查询匹配交易，可以帮助您对行情有精准的判断

委员选举
Committee Election

在委员选举窗口，您可以支付一定金额竞选成为系统中的委员 由委员构成的委员会将负责撮合系统中其他用户的挂单，达成匹配的交易 成为委员撮合交易将为您带来额外的收益，同时您也要保证自己按照规则进行撮合，否则将受到其他用户的举报，接受惩罚

点击右侧按钮，即刻注册账号使用
